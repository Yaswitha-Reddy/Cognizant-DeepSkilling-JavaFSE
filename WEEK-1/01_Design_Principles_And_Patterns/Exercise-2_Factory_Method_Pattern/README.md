# Exercise 2: Implementing Factory Method Pattern

## Aim
Implement the Factory Method Pattern to create different types of documents.

## Pattern Type
Creational Design Pattern

## Description
The Factory Method Pattern provides an interface for creating objects while allowing subclasses to decide which object to instantiate.

## Concepts Covered

- Abstract Classes

- Inheritance
- Polymorphism
